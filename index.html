Johnny-five.io

Johnny-Five is the JavaScript Robotics & IoT Platform.

An easy way for controlling hardware inspired by the web

Inspired by the web, there are events, callbacks.

The name is a reference to

||press down||

fictional robotic star of the movie Short Circuit

“Hello world”

						
								const { RaspiIO } = require('raspi-io');
								const five = require('johnny-five');
								
								const board = new five.Board({
									io: new RaspiIO()
								});
								
								board.on('ready', () => {
									const led = new five.Led('P1-12');
									led.blink();
								});
  • we import the libraries
  • and the highlited code does the magic
  • it creates a new led object
  • sets the correct pin
  • --- DOWN for pins ---
  • P1 is specific for the Raspberry Pi, it's header 1
  • 12 is the actual pin
